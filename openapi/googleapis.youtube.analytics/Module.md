## Overview
The Youtube Analytics API v2 connector(https://developers.google.com/youtube/analytics) OpenAPI specification.
The Youtube Analytics API provides the capability to retrieve your YouTube Analytics data.

### Key Features

- Programmatic access to create and manage resources via REST API
- Access analytics, reports, and performance metrics
- Manage user accounts and profiles
- Secure authentication with API key or OAuth support

## Prerequisites
Before using this connector in your Ballerina application, complete the following:
* Create a [Google account](https://accounts.google.com/signup)
* Obtain tokens - Follow [this link](https://developers.google.com/identity/protocols/oauth2)
