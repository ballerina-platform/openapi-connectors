## Overview
The GoDaddy Abuse API v1 connector(https://developer.godaddy.com/doc/endpoint/abuse) OpenAPI specification.

GoDaddy’s Abuse API is provided to simplify and standardize the abuse reporting experience.

### Key Features

- Programmatic access to create and manage resources via REST API
- Access analytics, reports, and performance metrics
- Manage user accounts and profiles
- Secure authentication with API key or OAuth support

## Prerequisites
Before using this connector in your Ballerina application, complete the following:

* Create an GoDaddy account
* Obtain tokens
    1. Follow [this link](https://developer.godaddy.com/getstarted) to obtain tokens
