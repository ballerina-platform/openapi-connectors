
Connects to [Owler API](https://corp.owler.com/) from Ballerina

### Package overview

The `owler` is a [Ballerina](https://ballerina.io/) connector for connecting to Owler platform.

#### Compatibility
|                       | Version                  |
|-----------------------|--------------------------|
| Ballerina Language    | Balerina Swan Lake Beta6 |
| Owler API             | v1.0                     |
