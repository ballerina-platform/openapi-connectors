Connects to [Workday Business Process API v1](https://community.workday.com/sites/default/files/file-hosting/restapi/index.html) from Ballerina.

### Package Overview

The `ballerinax/workday.businessprocess` is a [Ballerina](https://ballerina.io/) connector for Business Process API.  
This package provides the capability to easily access information on event actions and details.

#### Compatibility
|                               | Version                    |
|-------------------------------|----------------------------|
| Ballerina Language Version    | Ballerina Swan Lake 2201.0.0 |
| API Version                   | v1                         |
